# vue-music

> 一款vue版本的音乐播放器

# Music：

* vue2.0
* vuex
* vue-router
* axios
* less

# Usage

* 项目根目录安装依赖

npm install

* 浏览器访问localhost:8000

npm run dev

* 打包编译生成静态资源

npm run build
